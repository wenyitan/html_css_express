This is a paragraph whose visibility will change upon toggling of the button below. The way it works is by using JavaScript to control the visibility of the paragaph. This is the first time Recruit Wen is doing this:
The paragraph is assinged an id="Harry Potter" while the toggle button has id assigned as "invisibility cloak". I have defined a class called "cloaked" in the style.css file which sets the display of the paragraph to none.
I am using document.getElementById() (defined in script.js) to retrieve the html elements to be manipulated via JavaScript. The button checks for whether "cloaked" is in the list of classNames of the paragaph using paragraph.classList.
If it is, it is removed and the text of the button is changed to "cloak". If it is not, "cloaked" is added to the classList, which then applies the display defined in style.css to the paragraph and hides it.
